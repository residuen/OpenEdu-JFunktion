OpenEdu-JFunktion is an open-source tool, to solve and plot functions.
It has resemblance to a freeware windows program for curve sketching.
This program is written in Java

For more information, see the website at http://residuen.github.com/OpenEdu-JFunktion/

To report a bug or a make a suggestions, mail to:
kbay@gmx.de

INSTALLATION
Java 1.6 or higher has to be installed.

CREDITS
JFunktion is an open source project, supported presently by me.

"JFunktion" uses Java 1.6, the ScriptEngine with JavaScript as solver, and JFreeChart to plot the linecharts

Most used icons are from http://tango.freedesktop.org/Tango_Icon_Library
